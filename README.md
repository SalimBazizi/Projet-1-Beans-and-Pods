# ☕ Tableau de Bord Analytique - Beans & Pods (Projet 1)

## 📖 Contexte du Projet

L'objectif principal est de développer une application interactive en utilisant Streamlit pour analyser les données de ventes de l'entreprise **Beans & Pods** Cette analyse a été commanditée par la firme de capital-risque **Angeli VC** afin de mieux comprendre les habitudes d'achat et d'orienter une nouvelle campagne de marketing ciblée

## 🎯 Livrables Inclus

- **Application Streamlit** : Exploration et visualisation interactive du jeu de données.
- **Étude Statistique Descriptive** : Analyse complète des modèles et tendances de vente.
- **Rapport de Recommandations** : Stratégies marketing et suggestions de collecte de données futures, vulgarisées pour les propriétaires de l'entreprise.
